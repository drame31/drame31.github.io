About

Builder,notjustcoder.

Computer Engineering student at Universidad Hispanoamericana in Costa Rica. I'm in my element when I'm thinking about how things connect — how a request travels through a system, where the bottleneck is, why the API is slow on the third call but not the first.

I build backend-heavy full-stack apps. My go-to stack is FastAPI on the backend with React on the front, but I've shipped Flask apps, CLI tools, and a restaurant website for a real client. The tool choice follows the problem, not the other way around.

Most of my projects started because something was annoying. QueuePilot came from watching people stand in lines at a place that didn't need to run that way. HireFlow came from my own job hunt spreadsheet breaking down. The best software I've written has been for problems I was actually in.

Right now I'm interested in real-time systems, performance optimization, and building things that don't require a manual to use. Open to remote roles — full-stack, backend, or anything that makes me think harder than I did yesterday.
